# Projeto de Extensão II - ADS
## Website Oficial: Banda Trápala 🎸

Este projeto foi desenvolvido como parte do Componente Curricular de **Projeto de Extensão II** do curso de Análise e Desenvolvimento de Sistemas.

### 🎯 Objetivo
O objetivo principal é utilizar a tecnologia para a **Ação e Difusão Cultural**, promovendo a visibilidade e facilitando o gerenciamento de agenda da banda **Trápala**, sediada em Canoas/RS.

### 🛠️ Tecnologias Utilizadas
- **HTML5**: Estruturação semântica do conteúdo.
- **CSS3**: Estilização e design responsivo (Mobile-first).
- **Git & GitHub**: Controle de versão e deploy (GitHub Pages).
- **Integração**: Player do Spotify e API do WhatsApp.

### 🚀 Funcionalidades
- [x] Landing Page Responsiva.
- [x] Página "Sobre" detalhada.
- [x] Player de música integrado.
- [x] Agenda de shows atualizada.
- [x] Botão de contato direto para orçamentos.

### 📄 Documentação (ADS)
Este projeto aplica conceitos de:
- **Engenharia de Requisitos**: Levantamento de necessidades junto aos músicos.
- **IHC (Interação Humano-Computador)**: Foco na usabilidade e acessibilidade.
- **Gerência de Projetos**: Fluxo de trabalho via Git.
